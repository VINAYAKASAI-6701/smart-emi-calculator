== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the shortcode `[smart_emi_calculator]` in any post or page.

== Usage ==

== Frequently Asked Questions ==

= How do I style the form? =
You can add custom CSS from your theme or use a child theme. The form uses standard HTML tags for easy targeting.

= Can I use it on multiple pages? =
Yes, you can place the shortcode anywhere, as many times as you want.

= Does this plugin store any data? =
No. It's a pure frontend calculator. No database entries are saved.
